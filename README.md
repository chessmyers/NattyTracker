# README

## Set-up: 
```bash 
  git clone https://github.com/chessmyers/NattyTracker 
  cd NattyTracker
  bundle install && yarn install 
  ruby /bin/webpack
  rails s
``` 
go to localhost:3000 or 127.0.0.1:3000
